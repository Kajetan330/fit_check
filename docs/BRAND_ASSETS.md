# ByTaste Brand Assets

Project-owned vector assets are included in `public/brand/`.

## Files

- `public/brand/bytaste-mark.svg`: square app/social mark.
- `public/brand/bytaste-wordmark.svg`: horizontal wordmark.
- `public/icon.svg`: browser/PWA icon derived from the mark.

## Brand Decisions

- Primary ink: `#1B1418`
- Paper: `#F5F1F0`
- Accent: `#7C2237`
- Blush panel wash: `#E9DCDC`
- Stone text: `#8B8084`
- Tone: premium, practical, creator-first, editorial, not luxury-gatekept.

## Media Assets

The current app uses generated original placeholder media in `public/assets/media/` instead of external stock URLs. These assets are suitable for development, investor demos, and layout testing.

Before public launch, replace placeholder people/customer images with creator-provided or professionally licensed photos and collect written media releases.
